## 云锁支持哪些系统？

**服务器端**

**Windows:**

Windows Server 2003 SP2（x86/x64）

Windows Server 2008（x86/x64）

Windows Server 2012（x64）

**Linux:**

RedHat 5.0~RedHat 5.11（x86/x64）

RedHat 6.0~RedHat 6.6（x86/x64）

RedHat 7.0（x86/x64）

CentOS 5.0~CentOS 5.11（x86/x64）

CentOS 6.0~CentOS 6.6（x86/x64）

CentOS 7.0（x86/x64）

**PC客户端**

Windows XP SP3 及以上
